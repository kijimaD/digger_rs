## Feature

- Roguelike
- Keyboard Focused
- Symol encounter & Front view battle

## Operation

| Key       | Effect           |
|-----------|------------------|
| `Enter`   | OK               |
| `ESC`     | Cancel           |
| `w/a/s/d` | Movement         |
| `Space`   | Stepping         |
| `g`       | Get item         |
| `i`       | Open inventory   |
| `t`       | Drop item        |
| `r`       | Remove equipment |
| `/`       | Cheat menu       |

## Development

```shell
$ make build # development build

$ make docker
$ make release # release build

$ make fmt # format
```
