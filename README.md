## Feature

- Roguelike
- Keyboard Focused
- Symol encounter & Front view battle

## operation

- WASD: Movement
- G: Get item
- I: Open inventory
- T: Drop item
- R: Remove equipment

## Development


```shell
$ make build # development build

$ make docker
$ make release # release build

$ make fmt # format
```
